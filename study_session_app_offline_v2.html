How to use this app
  1. Import your timetable: click Import timetable HTML and pick your file. “Today” fills automatically.
  2. Enable alarms: press Enable sound in the Timer. Allow Notifications for banners.
  3. Start: pick a session in Today/Upcoming → hit Start. Stages: pre‑test → study → retrieval → error log → flashcards.
  4. Feynman notes: type your explanation or Upload notebook photos (compressed locally) if you wrote by hand.
  5. Flashcards: add atomic cards; Export Anki .tsv (Front | Back | Tags).
  6. Spaced reviews: finishing a session adds +1/+3/+7/+14 day review items.
  7. Mobile: bottom controls stick for Start/Pause/Skip; camera uses rear lens.
